・firebaseプロジェクトのデプロイ先について

現時点では、デプロイ先は本番環境となっています。
デプロイ先の変更は下記のコマンドにて行っております。


・Dev環境
firebase use drone-star-training-dev


・本番環境
firebase use drone-star-training-web


また、デプロイ先に合わせて、一部のソースを変更する必要があります。


[Proj_DroneSim]/Assets/Plugins/FirebasePlugin.jslib

85行目: FB_InitializeAppJS: function ()

// develop
からの行が、Dev環境でのfirebaseSDK設定で

// master
からの行が、本番環境でのfirebaseSDK設定になります。

デプロイ先に合わせて、コメント＆コメントアウトしてビルドしてください。
